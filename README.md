## 📥 安装

```sh
git clone https://github.com/Maicius/QQZoneSwipper.git
```

## 📤 部署

运行 deploy.sh (在命令行中输入 `./deploy.sh`) 或者将如下两个命令输入命令行: 

```sh
> npm install --save
> npm run start
```
