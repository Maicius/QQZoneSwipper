## 📥 Install

```sh
git clone https://github.com/Maicius/QQZoneSwipper.git
```

## 📤 Deploy

just run deploy.sh (enter`./deploy.sh` into command line) or enter the two commands below into shell: 

```sh
> npm install --save
> npm run start
```
